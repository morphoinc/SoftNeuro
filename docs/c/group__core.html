    Enumeration Type Documentation

    morapi_DType

    enum morapi_DType

    Data type.

    Each bit represents the followng.

    31-2827-2423-2019-1615-1211-87-43-0Description
    xxxxxxxxdata size.
    xxxxunsigned (0) or signed (1).
    xxxxfixed (0), float (1) or quantized (3).
    xxxxnormal (0) or extended (1).

    For example, MORAPI_DTYPE_FLOAT16 (= 0x00001102) means

    • data size: 2
    • signed (1)
    • float (1)
    • normal (0)
