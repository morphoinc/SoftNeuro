    morapi_ImageFormat

    enum morapi_ImageFormat

    Image format.

    • MSB <--------------------------—> LSB
    • 1111 0000 0000 0000 0000 0000 0000 0000 [Color space] 0(None), 1(Gray), 2(RGB), 3(RGBA), 4(YUV), 5(Bayer)
    • 0000 1111 0000 0000 0000 0000 0000 0000 [Pixel size in Plane 0.]
    • 0000 0000 1111 0000 0000 0000 0000 0000 [Pixel size in Plane 1.]
    • 0000 0000 0000 1111 0000 0000 0000 0000 [Pixel size in Plane 2.]
    • 0000 0000 0000 0000 1111 0000 0000 0000 [Pixel size in Plane 3.]
    • 0000 0000 0000 0000 0000 1111 1111 0000 [The order of pixel components.]
    • 0000 0000 0000 0000 0000 0000 0000 1111 [Optional bits]
