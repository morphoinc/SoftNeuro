    morapi_ByteArray_init()

    morapi_Result morapi_ByteArray_init (morapi_ByteArray * self,
                                     morapi_Size i_alloc_size,
                                     void * i_data)

    Initializes a byte array.

    If i_data is not NULL, allocation size and the memory are fixed to i_alloc_size and i_data respectively. Note that the size of the byte array is always 0 just after morapi_ByteArray_init(). If you want to have the byte array as a shallow copy of i_data, calls morapi_ByteArray_borrow(). If i_data is NULL, required memory is allocated internally as necessary.
